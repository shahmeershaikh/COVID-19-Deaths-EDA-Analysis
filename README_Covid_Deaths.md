# COVID-19 Deaths EDA Analysis

## Personal Motivation
This project is dedicated to my uncle and my friend's parent, who both lost their lives to COVID-19. This analysis is my way of understanding the impact of the pandemic and honoring their memory.

## Project Overview
A comprehensive exploratory data analysis of COVID-19 deaths and vaccinations using SQL and Tableau visualizations to uncover key insights about the pandemic's global impact.

## Key Analyses
- **Death Rate Analysis**: Likelihood of dying from COVID-19 by country
- **Infection Rates**: Percentage of population infected across countries
- **Highest Impact**: Countries with highest infection and death counts
- **Continental Breakdown**: COVID-19 impact by continent
- **Global Trends**: Daily and cumulative global case and death statistics
- **Vaccination Impact**: Vaccination rollout progress and population coverage

## Tools & Techniques
- **SQL**: CTEs, Window Functions, Joins, Temp Tables, Views, Aggregate Functions
- **Visualization**: Tableau dashboards for data storytelling
- **Data**: Global COVID-19 cases, deaths, and vaccination data

## Files
- `EDA_Covid19_Project.sql` - Main exploratory analysis queries
- `EDA_Project_Visualization.sql` - Queries optimized for Tableau
- `EDA_Covid19_Tableau_Data_visualization` - Interactive dashboards
